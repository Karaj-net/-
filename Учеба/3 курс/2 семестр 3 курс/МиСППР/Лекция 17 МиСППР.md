# Лекция 17
## Рекомендации по выбору и по применению методов принятия решений

На разных этапах процесса принятия решений (ППР) могут быть использованы различные методы. В связи с этим возникает задача: как правильно выбрать метод ПР для конкретной ситуации?

### Задача выбора метода принятия решений

Задача выбора метода может быть сформулирована следующим образом: **как, располагая определенными ресурсами и рядом доступных методов, определить тот метод, который может дать наилучший результат к требуемому моменту времени?**

Выбор метода проводится на основе следующих пяти ключевых требований:

1. **Практическая применимость метода.** Методы эффективны лишь при определенных условиях, при этом их использование возможно только при соблюдении конкретных ограничений.
2. **Стоимость использования метода.** Затраты на реализацию метода должны обязательно сравниваться с экономическим или управленческим эффектом, получаемым в результате его применения.
3. **Эффективность применения метода.** При анализе эффективности необходимо учитывать масштаб проблемы: в случаях, когда от решения предполагается получить большую отдачу, целесообразно использовать более дорогие, но более точные методы.
4. **Достоверность рекомендации.** При оценке достоверности необходимо знать, насколько крупные ошибки в данном решении являются допустимыми. Однако при использовании эвристических (неформальных) методов редко удается точно установить достоверность рекомендации до получения фактических результатов.
5. **Стабильность решения.** Данный показатель означает продолжительность (устойчивость) использования решения, полученного при помощи выбранного метода.
6. **Сбалансированность метода с другими методами.** Требует использовать для разных этапов формирования решения совместимые между собой методы. При этом общее качество решения предопределяется наименее эффективным этапом его формирования.

> **Важное правило:** Обычно при применении ПР невозможно выполнить в полном объеме абсолютно все предъявляемые к ним требования. Только обеспечив базовую *применимость* и *сбалансированность* метода, следует переходить к рассмотрению остальных условий отбора.

### Организация экспертных процедур и рекомендации по применению

При выборе метода на основе указанных требований следует учитывать множество аспектных факторов:
* Содержание и значение решаемой проблемы;
* Степень риска;
* Достоверность имеющихся данных;
* Затраты времени на решение проблемы;
* Существующие внешние и внутренние ограничения.

Экспертные оценки являются важнейшей информацией для ЛПР, необходимой для принятия взвешенных и обоснованных решений в преимущественно сложных, слабоструктурированных ситуациях ПР.

#### Основные этапы организации экспертных процедур на практике:
1. Проведение предварительных бесед.
2. Предварительный анализ проблемной ситуации.
3. Составление полного перечня альтернатив.
4. Выявление критериев оценки альтернатив.
5. Получение необходимой качественной и количественной информации.
6. Непосредственное применение выбранного метода принятия решений.
7. Предоставление и фиксация этапов и результатов решения.
