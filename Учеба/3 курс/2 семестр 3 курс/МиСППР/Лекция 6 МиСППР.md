# Лекция 6
## Нелинейное программирование (НЛП). Метод штрафных функций. Градиентные методы решения задач НЛП.

В общем случае в задачах математического программирования целевая функция или ограничения могут быть заданы нелинейными численными значениями. Для решения задач нелинейного программирования (НЛП) было предложено много различных подходов, но, в отличие от линейного программирования (ЛП), здесь **не существует единого универсального метода**. 

Выбор конкретного метода определяется содержанием решаемой задачи и опытом исследователя. В подавляющем большинстве случаев методы решения задач НЛП характеризуются как многошаговые методы или как **методы последовательного улучшения исходного (начального) решения**.

### Сложности при численной реализации методов НЛП

При расчете задач НЛП могут возникнуть следующие основные трудности:
1. **Проблема выбора подходящих начальных значений переменных.** Это связано с многоэкстремальным характером нелинейных функций. Если при разных начальных точках получается один и тот же экстремум — можно надеяться, что был найден *глобальный экстремум*.
2. **Снижение точности и накопление ошибок** в ходе вычислительной процедуры.

Каждый метод оценивается по следующим свойствам:
* Точность решений;
* Скорость сходимости;
* Время счета;
* Устойчивость метода.

### Классификация методов НЛП

Классификацию можно провести по различным признакам:
1. **По постановке задачи:** 
   * Условные (с ограничениями);
   * Безусловные (без ограничений);
   * Дискретные (целочисленные);
   * Задачи выпуклого, квадратичного, сепарабельного программирования.
2. **По характерным признакам алгоритма:** 
   * Градиентные методы;
   * Безградиентные методы;
   * Детерминированные методы;
   * Методы случайного поиска;
   * Комбинированные методы.

> **Важный вопрос:** Можно ли использовать методы решения задач ЛП при решении задач НЛП? В принципе, это возможно (путем линеаризации), но на практике таким подходом почти никто не пользуется.

### Метод штрафных функций

Если в задаче присутствуют условия-ограничения, то она является *условной*. Безусловные задачи НЛП решаются значительно легче, чем условные задачи оптимизации. 

Поэтому при решении условных задач НЛП, наряду с прямыми методами, часто применяется метод преобразования условной задачи в эквивалентную ей **последовательность задач безусловной оптимизации** путем введения так называемых *штрафных функций* $\psi(x)$. Штрафная функция равна $0$ внутри допустимой области пространства проектирования и резко возрастает при нарушении ограничений.

#### Алгоритм решения задачи с использованием штрафных функций:
1. Выбирается начальная точка $x_0$ и монотонно убывающая, сходящаяся к нулю последовательность положительных чисел $r_k$ (например, $r_k = r_{k-1} / 2$ или $r_k = r_0 / q^{k-1}$, где $q > 0$).
2. При $k = 1, 2, 3 \dots$, начиная с точки $x_{k-1}$ (полученной на прошлой итерации), решается задача безусловной оптимизации/минимизации модифицированной функции $F(x, r_k)$.
3. В результате решения определяется очередная точка $x_k$. Если на каждом шаге алгоритма удается найти глобальный минимум функции $F(x, r_k)$ по $x$, то последовательность точек $x_k$ будет строго сходиться к глобальному минимуму исходной функции $f(x)$ при условии $r_k \to 0$.

### Градиентные методы решения задач НЛП

Применение классических методов математического анализа (через производные) для нахождения экстремума функции в реальных задачах связано с большими трудностями:
* Классические формулы дают только *необходимые*, но не достаточные условия экстремума;
* Приходится решать сложные системы нелинейных уравнений;
* Искомый экстремум может оказаться на самой границе допустимой области.

Одним из наиболее эффективных численных методов решения задач НЛП являются **градиентные методы**.

#### Метод наискорейшего спуска (крутого восхождения):
Согласно данному методу, выбирается начальная точка $x_0$ и совершается расчетный шаг по направлению градиента целевой функции (или против него при минимизации). Если при этом значение функции $f(x)$ улучшилось, движение по этому же направлению продолжается до тех пор, пока не будет найден локальный экстремум в данном направлении. 

Затем в полученной точке $x_1$ заново вычисляется градиент и совершаются следующие аналогичные шаги. Данный метод позволяет значительно быстрее найти экстремум функции, чем стандартный покоординатный подбор.
