# Лекция 9
## Методы многокритериальной оптимизации. Линейная свертка критериев. Принцип Парето.

Возникновение в задаче нескольких противоречивых критериев (или единого векторного критерия) порождает **две основные математические задачи**:
1. Построение процедур выявления предпочтений ЛПР на языке векторных оценок альтернатив (**задача многокритериальной оптимизации**).
2. Построение процедур оценивания и отсеивания «плохих» альтернатив (выделение подмножества альтернатив, максимальных по данной системе критериев).

Поскольку неопределенность целей носит неустранимый характер, для однозначного выбора компромиссного решения требуется привлечение дополнительных гипотез и специальных подходов.

### Основные подходы к решению задач многокритериальной оптимизации

Выделяют 4 базовых класса процедур для работы с векторными критериями:

1. **Априорные процедуры** — методы приведения многокритериальных задач к однокритериальным с помощью предварительного компромиссного выбора решающего правила.
2. **Построение множества оптимальных по Парето альтернатив** — выделение области эффективных решений, из которой ЛПР на основе своего опыта осуществляет наиболее предпочтительный выбор.
3. **Апостериорные процедуры** — аксиоматический подход на базе теории полезности для построения функции ценности после первичного анализа вариантов.
4. **Адаптивный подход** — интерактивная человеко-машинная процедура для поэтапного выявления предпочтений ЛПР одновременно с исследованием множества доступных альтернатив.

---

### Априорные процедуры: Линейная свертка критериев

Сталкиваясь с многокритериальной оптимизацией, исследователи естественно пытаются свести её к классической однокритериальной задаче, для которой существуют точные численные методы решения. 

Самым простым и распространенным методом является **линейная свертка**. При данном подходе формируется один суперкритерий путем умножения каждого частного критерия на свой весовой коэффициент важности с последующим их суммированием.

* **Главная сложность метода:** Правило свертывания множества критериев в один может быть выбрано ЛПР по-разному. Изначально строго математически неизвестно, какой из вариантов весовых коэффициентов является истинно предпочтительным.

---

### Принцип Парето

Если свертка критериев невозможна или нежелательна, используют **принцип Парето** — один из важнейших принципов объективного отбора управленческих решений. 

> **Множество Парето (множество эффективных альтернатив)** — это такое подмножество вариантов, которое обладает следующим свойством: ни одно из решений внутри него нельзя улучшить по какому-либо одному критерию без одновременного ухудшения его качества по другим критериям.

По принципу Парето группа (или система) не может улучшить свои показатели без нанесения прямого ущерба хотя бы одному из частных критериев или интересов участников. Его применение на практике возможно только при сильной взаимозависимости всех элементов системы. 

#### Особенности применения множества Парето:
* **Информативность для ЛПР:** Выделение множества Парето дает руководителю колоссальную информацию. ЛПР наглядно видит, сколько «стоит» (какой ценой дается) увеличение одного из целевых показателей и как именно оно сказывается на остальных показателях, значения которых непременно ухудшаются.
* **Роль ЛПР:** Окончательный выбор конкретного компромиссного варианта из паретовского множества всегда остается за человеком. Помимо частного вектора критериев $f(x)$, в распоряжении ЛПР часто бывают некоторые общие интегральные критерии $F(x)$ (например, общая итоговая стоимость проекта или суммарный доход организации). В таких случаях, чтобы найти финальное решение, поверх множества Парето ставят задачу максимизации общего показателя: $\max F(x)$.
* **Численное построение:** На практике множество Парето строится приближенными численными методами. Пусть имеется некоторое исходное множество параметров $G_x$, тогда порожденное им новое пространство критериальных оценок $G_f$ называется *множеством достижимости*.
