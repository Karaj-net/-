# Лекция 16
## Экспертные системы ПР. Экспертные оценки, организация экспертных процедур (основные этапы)

При невозможности или сложности использования строгих математических методов применяются неформальные методы ПР, то есть **экспертные методы**. Примером традиционных эвристических процедур являются различные экспертизы. 

Экспертная система положила начало развитию совокупности методов инженерии знаний, составляющих новый подход к созданию высокоэффективных программных систем. Технологию построения экспертных систем называют инженерией знаний. Важным свойством таких систем является то, что их можно использовать для обучения, тренировки руководящих работников и ведущих специалистов.

### Основные участники в работе экспертных систем

В «игре» (работе) экспертных систем выделяют следующих участников:
1. **Сама экспертная система** — набор программ или программное обеспечение, которое решает задачи в интересующей предметной области (содержит компоненту, решающую проблему, и поддержку).
2. **Эксперт** — человек, способный ясно выражать свои мысли и пользующийся репутацией специалиста, умеющего находить правильное решение проблем конкретной предметной области.
3. **Инженер знаний** — человек, как правило, имеющий знания в области информатики, искусственного интеллекта (ИИ) и знающий, как надо строить экспертные системы.
4. **Средство построения экспертных систем** — языки программирования и инструментальные средства, используемые для создания системы.
5. **Пользователь** — человек, который использует уже построенную экспертную систему.

Правила построения экспертных систем во многих случаях являются эвристическими, то есть нематематическими. Выделенные знания по предметной области называются **базой знаний**, а общие знания о нахождении решения задач — **механизмом вывода**.

Различают 2 уровня использования экспертных суждений:
1. Качественные
2. Количественные

### Методы и этапы экспертного опроса

В связи с ограниченными возможностями применения математических методов экспертная оценка часто является единственным средством решения многих задач. На практике применяются как индивидуальные, так и групповые (коллективные) экспертные опросы.

* **Метод Дельфи** — один из эффективных методов использования суждений экспертов, предусматривающий проведение опроса в несколько туров. Его основные предпосылки:
    1. Поставленные вопросы должны допускать возможность выражения ответа в виде чисел.
    2. Эксперты должны обладать достаточной информацией для того, чтобы дать оценку.
    3. Ответ на каждый из вопросов (оценка) должен быть экспертом обоснован.
* **Метод коллективной генерации идей (мозговая атака)** — применяется при групповых оценках. Обычно используется группа из 10–15 человек (число может быть доведено до 100).

#### Основные этапы экспертного опроса:
1. Подбор экспертов и формирование экспертных групп.
2. Формирование опросов и составление анкет.
3. Формирование правил определения суммарных оценок на основе оценок отдельных экспертов.
4. Работа с экспертами.
5. Анализ и обработка экспертных оценок.

Ценность полученных оценок зависит от опыта и интуиции лиц, формулирующих оценку. Единодушие большинства экспертов не всегда является критерием достоверности. К недостаткам экспертных систем относится отсутствие достаточной устойчивости и опасность систематического занижения прогнозных оценок заинтересованными сотрудниками. Одним из важнейших показателей эксперта является его **надежность**.

### Принятие решений на основе групповых предпочтений

Для образования единого группового предпочтения $F = F(f_1, f_2, ..., f_d)$ необходимо согласовать индивидуальные предпочтения членов группы. Конкретные принципы согласования:
1. **Принцип большинства голосов** (более 1/2).
2. **Принцип Курно** — отражает индивидуальную рациональность, когда все коалиции являются одноэлементными. Никому из членов группового ЛПР невыгодно менять решение, так как для него индивидуально не существует лучшего варианта.
3. **Принцип Парето** — применяется, когда все члены ЛПР образуют единую коалицию. Группа не может улучшить свое решение без нанесения ущерба хотя бы одному ее члену.
4. **Принцип Эджворта** — обобщает принципы Курно и Парето для случаев произвольного числа коалиций. Каждой коалиции невыгодно менять свои решения, так как нет лучшего варианта.

### Критерии оценки альтернатив в экспертных системах

В результате работы экспертов получаются экспертные оценки (количественные или порядковые). Для оценки альтернатив по платежной матрице используются следующие критерии:

1. **Минимаксный / максиминный критерий (Критерий Вальда)** — выражает наибольшую осторожность и полностью исключает риск:
   $$U^*_{min} = \max_i \min_j U_{ij}$$
2. **Критерий максимакса (крайнего оптимума)**:
   $$U^* = \max_i \max_j U_{ij}$$
3. **Критерий Сэвиджа (минимаксного риска)** — минимизирует максимальный риск:
   $$U^* = \min_i \left[ \max_j (\max U_{ij} - U_{ij}) \right]$$
4. **Критерий Гурвица (обобщенного максимина)** — наиболее уравновешенная позиция пессимизма-оптимизма при $0 \le \alpha \le 1$:
   $$U^* = \max_i \left[ \alpha \cdot m_i + (1 - \alpha) \cdot M_i \right]$$
5. **Критерий Лапласа** — предполагает равную вероятность исходов:
   $$U^* = \max_i \left( \frac{\sum_{j=1}^{m} U_{ij}}{m} \right)$$

### Области применения экспертных систем
Основное назначение состоит в решении неформализованных задач в следующих сферах:
* Диагностика
* Ремонт
* Обучение
* Управление
* Военные приложения
* Здравоохранение
* Охрана правопорядка
