# События 
## Понятие события и их краткая форма
События при работе с полями типа делегат играют такую же роль, как свойства при работе с обычными полями, то есть это инструменты к доступу к полям. Отметим, говоря о свойствах мы имели автоматические свойства и развернутые свойства. В случае  с событиями мы также имеем 2 формы: краткую и развернутую. Рассмотрим краткую форму:
Здесь мы в классе описываем события по аналогии с описанием поля, добавляя ключевое слово event. Отметим в этом случае само поле описывать не обязательно, оно будет создано автоматически
```C#
class Account5
{
	private string name;
	private int sum;
	private string password;
	
	public event Action<string> Sob;
	public Account5()
	{
		name;
		sum;
		password;
	}
	
	void Add(int s)
	{
		Sob();
	}
	void Sub(int s)
	{
		sob -> Sob
	}
	public void Act()
	{
	
	}
}

```
Тот класс где описано событие называется издателем. Тот класс, где используется события называется подписчиком. += называется подпиской, а -= называется отпиской от события. Может быть несколько подписчиков на одно событие 
Обычно в программировании под событием принимают выполнение некоторого действия - нажатие на кнопку и т.д. В принципе, при любых таких действиях изменяются значение некоторого поля. Таким образом, происходит реакция на изменения этого значения
## Развернутая форма
Как и в случае свойств, здесь имеется два блока Add и Remove - они соответственно выполняются при подписке и отписке к событию
```C#
class Account6
{
	private string name;
	private int sum;
	private string password;
	
	private event Action<string> sob;
	public Account6()
	{
		name;
		sum;
		password;
	}
	
	void Add(int s)
	{
		Sob();
	}
	void Sub(int s)
	{
		sob -> Sob
	}
	public void Act()
	{
	
	}
	public event Action<string> Sob
	{
		add
		{
			Console.Writelne("Клиент {0}, введите пароль для добавления метода!",                               name);
			string pw = Console.Readline();
			if (pw == password)
			{
				sobf += value;
				Console.Writeline("Добавлен метод {0}", value.Method.name)
			}
			else
			{
				Console.Writeline("Неверный пароль")
			}
		}
		remove
		{
			Console.Writelne("Клиент {0}, введите пароль для удаления метода!",                               name);
			string pw = Console.Readline();
			if (pw == password)
			{
				sobf -= value;
				Console.Writeline("Удален метод {0}", value.Method.name)
			}
			else
			{
				Console.Writeline("Неверный пароль")
			}
		}
	}
}


class Program
{
	static void PrFile1()
	{}
	static void Main()
	{
		Account5 acc11 = new Account6("Иванов", 50, "123");
		acc11.Sob += Pr2Con;
		acc11.Sob += PrFile1;
		dcc11.Act();
		
		acc11.Sob -= PrFile1;
		acc11.Sob += PrFile2;
		acc11.Act();
	}
}
```
