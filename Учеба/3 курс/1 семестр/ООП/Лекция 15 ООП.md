# Делегат EventHandler
В классе аккаунт6 мы для обработки события мы использовали делегат `Action<strung>` в принципе получили рабочую программу, но желательно для обработки событий использовать делегат EventHandler - обработчик событий. Этот делегат имеет два параметра:

```C#
EventHandler(object a, EventArgs e)
```
а - объект, который вызвал событие. е - контейнер, пустой класс его нужно создать дочерний, который содержит информацию о данном событии, чем вызвано событие

таким образом мы должны описать дочерний класс, таким образом мы можем создать поля для старого и нового значения сумму при совершении события

class AccountEventArgs: EventArgs

	private int oldSum;
