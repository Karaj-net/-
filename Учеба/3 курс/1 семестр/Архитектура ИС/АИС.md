## Исправленная программа с использованием регистров

### Таблица программы:

| Адрес | Мнемокод | Код       | Примечание |
|-------|-----------|-----------|------------|
| 000   | IN        | 01 0 000  | Ввод X в аккумулятор |
| 001   | WR R1     | 22 0 001  | Сохранение X в регистр R1 |
| 002   | RD R1     | 21 0 001  | Загрузка X из R1 для проверки |
| 003   | SUB #24   | 24 1 024  | X - 20 (20₁₀ = 24₈) |
| 004   | JS 015    | 13 0 015  | Если X < 20, переход на адрес 015 |
| 005   | RD R1     | 21 0 001  | Ветка 1 (X ≥ 20): загрузка X |
| 006   | DIV #3    | 46 1 003  | X / 3 |
| 007   | WR R2     | 22 0 002  | Сохранение X/3 в R2 |
| 010   | RD R1     | 21 0 001  | Загрузка X |
| 011   | ADD R2    | 23 0 002  | X + X/3 |
| 012   | DIV #12   | 46 1 012  | /10 (10₁₀ = 12₈) |
| 013   | WR R2     | 22 0 002  | Сохранение результата Y в R2 |
| 014   | JMP 036   | 10 0 036  | Переход на вывод |
| 015   | RD R1     | 21 0 001  | Проверка: X < 2? |
| 016   | SUB #2    | 24 1 002  | X - 2 |
| 017   | JS 032    | 13 0 032  | Если X < 2, переход на ветку 3 |
| 020   | RD R1     | 21 0 001  | Проверка: X > 12? |
| 021   | SUB #14   | 24 1 014  | X - 12 (12₁₀ = 14₈) |
| 022   | JNS 032   | 13 0 032  | Если X > 12, переход на ветку 3 |
| 023   | RD R1     | 21 0 001  | Ветка 2 (2 ≤ X ≤ 12): загрузка X |
| 024   | ADD #3    | 23 1 003  | X + 3 |
| 025   | MUL #3    | 25 1 003  | (X+3) × 3 |
| 026   | WR R2     | 22 0 002  | Сохранение результата Y в R2 |
| 027   | JMP 036   | 10 0 036  | Переход на вывод |
| 032   | RD #0     | 21 1 000  | Ветка 3 (остальные случаи): Y = 0 |
| 033   | WR R2     | 22 0 002  | Сохранение Y = 0 в R2 |
| 034   | JMP 036   | 10 0 036  | Переход на вывод |
| 036   | RD R2     | 21 0 002  | Загрузка Y из R2 для вывода |
| 037   | OUT       | 02 0 000  | Вывод Y через устройство вывода |
| 040   | HLT       | 09 0 000  | Остановка программы |

### Пояснения к исправлениям:

1. **Использование регистров**:
   - R1 используется для хранения входного значения X
   - R2 используется для хранения промежуточных результатов и конечного значения Y
   - Все операции выполняются через регистры, что ускоряет выполнение программы

2. **Вывод через OUT**:
   - Команда OUT выводит значение из аккумулятора на устройство вывода
   - Перед OUT значение Y загружается из регистра R2 в аккумулятор

3. **Логика программы**:
   - Если X ≥ 20: Y = (X/3 + X)/10
   - Если 2 ≤ X ≤ 12: Y = (X+3)×3  
   - Иначе: Y = 0

4. **Оптимизация**:
   - Убраны лишние повторяющиеся команды
   - Улучшена читаемость кода
   - Сохранена правильная логика переходов между ветвями алгоритма
